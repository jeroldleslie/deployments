# neverwinter deployments

This project automates the installation and configuration of the following software components:

* ambari_agent
* ambari_server
* basho_bench
* HDP (blueprint & blueprint_execute)
* common
* druid
* grafana
* gripper
* influxdb
* java
* jmxtrans
* kafka_broker
* kafka_zookeeper
* shinken
* storm
* drill
